_XOPEN_REALTIME_THREADS
=======================

The ``_XOPEN_REALTIME_THREADS`` option corresponds to the
:ref:`XSI_REALTIME_THREADS <posix_option_group_xsi_realtime_threads>` option group.

Enable with :kconfig:option:`CONFIG_XSI_REALTIME_THREADS`. See the option group page for the
constituent POSIX options.
