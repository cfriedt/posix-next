_POSIX_CLOCK_SELECTION
======================

The ``_POSIX_CLOCK_SELECTION`` option corresponds to the
:ref:`POSIX_CLOCK_SELECTION <posix_option_group_clock_selection>` option group.

Enable with :kconfig:option:`CONFIG_POSIX_CLOCK_SELECTION`. See the option group page for per-API
detail.
