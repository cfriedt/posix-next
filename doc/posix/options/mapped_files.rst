_POSIX_MAPPED_FILES
===================

The ``_POSIX_MAPPED_FILES`` option corresponds to the
:ref:`POSIX_MAPPED_FILES <posix_option_group_mapped_files>` option group.

Enable with :kconfig:option:`CONFIG_POSIX_MAPPED_FILES`. See the option group page for per-API
detail.
