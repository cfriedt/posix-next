_POSIX_THREADS
==============

The ``_POSIX_THREADS`` option is synonymous with the
:ref:`POSIX_THREADS_BASE <posix_option_group_threads_base>` option group.

Enable with :kconfig:option:`CONFIG_POSIX_THREADS`. See
:ref:`POSIX_THREADS_BASE <posix_option_group_threads_base>` for per-API detail.
