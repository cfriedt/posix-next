POSIX_C_LIB_EXT
===============

Enable this option group with :kconfig:option:`CONFIG_POSIX_C_LIB_EXT`.

.. csv-table:: POSIX_C_LIB_EXT
   :header: API, Supported
   :widths: 50,10

    :c:func:`fnmatch`, yes
    :c:func:`getopt`, yes
    :c:func:`getsubopt`, yes
    optarg, yes
    opterr, yes
    optind, yes
    optopt, yes
    :c:func:`stpcpy`, yes
    :c:func:`stpncpy`, yes
    :c:func:`strcasecmp`, yes
    :c:func:`strdup`, yes
    :c:func:`strfmon`, yes
    :c:func:`strncasecmp`, yes
    :c:func:`strndup`, yes
    :c:func:`strnlen`, yes
