_POSIX_REALTIME_SIGNALS
=======================

The ``_POSIX_REALTIME_SIGNALS`` option corresponds to the
:ref:`POSIX_REALTIME_SIGNALS <posix_option_group_realtime_signals>` option group.

Enable with :kconfig:option:`CONFIG_POSIX_REALTIME_SIGNALS`. See the option group page for per-API
detail.
