POSIX API Search
################

Search across all documented POSIX symbols — functions, types, macros, and
structs — extracted from the Doxygen output.  Results are filtered in real
time as you type.  A blank search box shows nothing; start typing a symbol
name or fragment to begin.

Clicking a symbol name opens its full Doxygen documentation page.  The
**Standard** link takes you to the Open Group POSIX.1-2017 specification
for that symbol.
