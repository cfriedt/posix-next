_POSIX_MEMORY_PROTECTION
========================

The ``_POSIX_MEMORY_PROTECTION`` option corresponds to the
:ref:`POSIX_MEMORY_PROTECTION <posix_option_group_memory_protection>` option group.

Enable with :kconfig:option:`CONFIG_POSIX_MEMORY_PROTECTION`. See the option group page for
per-API detail.
